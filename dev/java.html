Осваиваем Java!

Курсы по изучению Java предназначены для тех, кто заинтересован в высокооплачиваемой работе. Карьерный рост вам обеспечен!

Java — универсальный язык программирования, предназначенный для разработки любого программного обеспечения. Полученные в результате использования Java приложения легко масштабируются и могут быть применимы на разных устройствах. Это один из наиболее защищенных языков, поэтому он востребован во всех бизнес-направлениях. Вы можете предложить свои профессиональные услуги банкам, медицинским центрам, гостиницам — любым организациям, в которых есть центры данных и необходимость в математических вычислениях. Хотите стать разработчиком игр? Java пригодится вам и в этом случае!

Что вы узнаете на курсе?

Курс состоит из теоретических и практических занятий, на которых изучаются основы программирования, в результате чего вы будете знать:

  • свойства алгоритма;
  • итеративные и рекурсивные алгоритмы;
  • типы данных и массивы;
  • все об управляющих операторах;
  • основы реляционной модели баз данных;
  • как установить сервер и клиента СУБД;
  • структуру данных в Java;
  • как обрабатывать ошибки и многое другое.

Перед вами откроются широчайшие перспективы программирования на языке Java — для повышения вашей востребованности как специалиста.

Программа

1-й модуль Основные понятия Java. JVM, JRE, JDK. Компиляция и запуск из консоли. Classpath. Intellij IDE. Установка ПО. Примеры кода программы c main - как скомпилировать и запустить.

2-й модуль Типы данных, литералы, преобразование между примитивами. Рассмотрение примеров на живом коде. Обёртки над примитивами.

3-4й модули Операции и операторы, приоритет, типы операндов в Java. Основные конструкции языка Java. Массивы и строки в Java. Решение алгоритмических задачек на Java.

5-й модуль Основы объектно-ориентированного программирования. Классы и объекты в Java. Класс Object. Модификаторы доступа. Наследование, инкапсуляция, полиморфизм. Примеры кода по всем 3-м принципам.

6-й модуль Оператор instanceof. Преобразования между классами, массивами. Перегрузка, перекрытие и сокрытие методов в Java. Ключевые слова final, static. Класс Class (предназначение, способы получения). GC.

7-й модуль Абстрактные классы и интерфейсы. Javadoc. Generics.

8-й модуль Исключения в Java.

9-й модуль Коллекции в Java. Enum.

10-й модуль Потоки ввода-вывода (IO): cимвольные, байтовые потоки, Scanner, Formatter, IO: надстройки. Сериализация/десериализация в Java.

11-12й модули Threads в Java.

13-14й модули Maven и Github

15-й модуль Финальный экзамен. Подготовка к собеседованию.
